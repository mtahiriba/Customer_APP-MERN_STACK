## How to Setup:
1. git clone git_url
2. open terminal
3. cd ./backend
4. add a new file in root named ".env"
5. add environment variables inside ".env" file. (reffer ".env.example" file from root folder)
6. npm install
7. npm run dev
8. open new terminal
9. cd ./frontend
10. npm install
11. npm run start
